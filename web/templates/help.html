Nifty Note
  • Help

Nifty Note is a simple, lightweight note-taking application. It provides a web-based and command line tool for managing text-based notes.

Features

  • Write plaintext or markdown. Math formulas are supported via MathJax.
  • Access your notes anywhere, from any device.
  • Use Supernifty's online application, or run locally by forking the GitHub repository.
  • Open source. Check out the code at GitHub.

    • Other Notes

  • The online version uses Google for authentication.
  • Use of the service is currently free. We may introduce a subscription model for more substantial usage. Though if this is an issue you can always run your own local version of the software.
